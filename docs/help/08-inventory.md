# 库存管理

库存管理提供实时库存视图、低库存预警、库存调整与全链路流水追溯。

## 库存视图

库存列表按「商品 → 型号」树形展示，每个节点包含：

- **在库数量**：实际库存总量
- **可用数量**：在库数量减去已锁定（待发货占用）数量
- 批次信息：各入库批次的数量与成本

支持按商品筛选，展开型号行查看批次明细。

## 库存预警

系统自动监控低库存商品：

- 预警阈值**优先使用商品型号上设置的最低库存**，未设置时回退到系统全局阈值
- 停用商品、停用/已删除的型号不参与预警计算
- 库存列表中低库存行会标红提示；打开「仅低库存」开关可只看预警商品

> 驾驶舱「库存预警」卡片与此口径完全一致，点击可直接跳转查看。

## 库存流水

每一次库存变动（销售发货、采购入库、退货、调整）都会生成流水记录，包含变动类型、数量、前后结存与关联单据，支持按商品/时间追溯。

## 库存调整

盘点发现账实不符时，通过库存调整单修正：

1. 库存调整列表点击「新建调整」
2. 选择商品/型号，填写调整数量（正数为盘盈，负数为盘亏）与原因
3. 提交后库存即时更新并生成流水

## 一致性校验

库存页右上角「一致性校验」用于核对库存结存与流水累计是否一致，建议在定期盘点后执行，发现差异及时排查。
