# 销售退货管理

已发货订单发生售后问题时，通过销售退货单处理退货与退款。

## 退货类型

| 类型 | 说明 | 库存影响 |
| --- | --- | --- |
| 仅退款 | 不退回货物，仅退还货款 | 不恢复库存 |
| 退货退款 | 货物退回并退款 | 退回商品恢复库存 |

> 退货类型由创建时选择决定，**创建后不可修改**，系统据此自动决定是否恢复库存。

## 创建退货单

1. 在订单列表对已发货订单点击「退货」
2. 选择退货类型
3. 勾选退货的明细行与数量（仅退款还需填写退款数量）
4. 填写退款金额与原因后提交

## 库存恢复

- 「退货退款」类型提交后，退回数量自动加回库存并生成库存流水
- 「仅退款」不影响库存

## 注意事项

- 只有**已发货**（部分或全部）的订单可以发起退货；未发货订单请直接取消
- 退货数量不能超过已发货数量
- 退款金额以实际协商结果填写，系统记录用于对账
