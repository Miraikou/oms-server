# 采购管理

采购管理覆盖向供应商补货的全过程：采购下单 → 到货入库 → 采购退货。

## 采购单状态

| 状态 | 含义 |
| --- | --- |
| 待入库 | 已下单，尚未到货 |
| 部分入库 | 部分明细已入库 |
| 全部入库 | 全部明细入库完成 |
| 已关闭 | 手动关闭，不再跟踪 |

## 创建采购单

1. 采购单列表点击「新建采购单」
2. 选择供应商、币种（默认 CNY）、汇率、采购日期
3. 添加采购明细：商品、型号、数量、单价
4. 保存后采购单进入「待入库」

> 仅「待入库」状态的采购单允许修改；已入库的单据不可再编辑。

## 入库

1. 对目标采购单点击「入库」
2. 填写本次到货的明细数量（可分批入库）
3. 提交后**库存增加**，生成对应库存批次与流水

部分入库后状态变为「部分入库」，全部到货后变为「全部入库」。

## 采购退货

供应商来货有质量问题时：

1. 在采购单操作列点击「退货」
2. 选择退货明细与数量，填写原因
3. 提交后**扣减对应库存**并生成流水

## 关闭采购单

长期未到货或作废的采购单可执行「关闭」，关闭后不参与待入库统计。

## 与驾驶舱的联动

驾驶舱「待入库采购」卡片统计「待入库 + 部分入库」的采购单，点击卡片自动跳转到对应筛选结果。
