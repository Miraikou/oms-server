# 系统管理

系统管理模块面向管理员，负责用户、权限、业务选项与系统设置。包含：用户管理、角色管理、菜单管理、字典管理、操作日志、登录日志、系统参数、帮助文档管理，以及个人信息页。

## 用户管理

> 【图片占位：用户管理列表页】

### 列表与搜索

搜索：账号/姓名（模糊匹配）、状态（启用/停用）。

列表字段：账号、姓名、手机号、邮箱、角色（标签展示）、状态、最后登录时间、创建时间。

### 新增用户

| 字段 | 说明 |
| --- | --- |
| 账号 | 必填，创建后不可修改 |
| 密码 | 必填，至少 6 位 |
| 姓名 | 必填 |
| 手机号 | 选填，11 位 |
| 邮箱 | 选填，需符合邮箱格式 |
| 角色 | 选填，可多选（决定该用户能看到哪些菜单和操作） |
| 状态 | 默认启用 |
| 备注 | 选填 |

> 账号不可重复。未分配角色的用户登录后将看不到任何菜单。

### 编辑用户

可修改姓名、手机号、邮箱、角色、状态、备注。**账号不可修改。**

> 编辑时如果清空角色选择，将移除该用户的全部角色（有意的安全管理操作）。

### 重置密码

点击操作列「重置密码」，直接设置新密码（至少 6 位）即可。重置后用户下次登录使用新密码。

### 停用影响

- 停用后该用户无法登录（提示「用户不存在或已停用」）
- 已登录的设备在登录状态到期后需要重新输入密码，等效强制下线
- 停用操作没有额外限制，请谨慎操作，避免误停用关键账号

## 角色管理

> 【图片占位：角色分配权限弹窗（菜单树勾选）】

### 基本概念

角色是权限的集合：给用户分配角色 → 角色关联菜单和按钮 → 用户获得对应的可见菜单与可操作按钮。

系统预置 6 个角色：超级管理员、老板、销售、采购、仓库、财务。

### 新增/编辑角色

**字段**：角色名称（必填，唯一）、角色编码（必填，唯一，创建后不可修改）、状态、备注。

### 分配权限

点击「分配权限」打开菜单树弹窗，勾选该角色可访问的菜单和按钮：

- 树节点包含目录、菜单、按钮三级
- 按钮节点显示权限码（如 `order:create`）
- 保存后**全量覆盖**该角色的权限配置

### 删除角色

已被用户引用的角色无法删除（提示「该角色已被用户引用，无法删除」）。需先解除所有用户的该角色关联。

### 超级管理员特权

超级管理员角色拥有特殊待遇：

- 自动拥有系统全部功能的访问权限，无需逐一分配
- 自动获得全部启用菜单和按钮权限
- 驾驶舱默认全局视图

## 菜单管理

> 【图片占位：菜单管理树形表格】

维护左侧导航的菜单树，以树形表格展示（无分页，默认全部展开）。

### 菜单类型

| 类型 | 说明 | 特有字段 |
| --- | --- | --- |
| 目录 | 一级分组（如「业务操作」「仓储采购」） | 页面地址、图标 |
| 菜单 | 具体页面（如「订单管理」） | 页面地址、图标 |
| 按钮 | 页面内的操作权限点 | **权限标识**（如 `order:create`） |

### 新增/编辑菜单

- 上级菜单：树形选择（按钮类型不能作为父节点）
- 菜单类型、菜单名称（必填）
- 按钮类型：只需填写权限标识
- 目录/菜单类型：填写页面地址（由系统实施人员提供）和图标
- 排序号、是否显示、状态

### 删除限制

- 有子菜单时无法删除（需先删除子节点）
- 被角色引用时无法删除（需先在角色中取消勾选）

### 权限码说明

按钮型菜单的「权限标识」是全系统权限控制的唯一来源。页面上的按钮会根据当前用户拥有的操作权限决定是否显示。如果用户看不到某个按钮，请检查其角色是否勾选了对应的按钮权限。

## 字典管理

维护系统动态枚举值，采用「字典类型 → 字典项」两级结构。

### 字典类型

搜索：编码/名称、状态。列表字段：字典编码、字典名称、状态、创建时间、备注。

**字段**：字典编码（必填，如 `PAYMENT_METHOD`，创建后不可修改，唯一）、字典名称（必填）、备注。

### 字典项

点击类型的「查看」进入字典项管理抽屉：

**字段**：字典值（必填，如 `ALIPAY`）、字典标签（必填，显示文本如「支付宝」）、排序号、状态、备注。

**唯一约束**：同一字典类型下字典值不可重复。

### 注意事项

- 字典类型和字典项**只能停用/启用，不能删除**
- 停用后业务页面的对应下拉选项自动隐藏
- 系统预置了 PAYMENT_METHOD（支付方式）和 TRADE_TYPE（交易方式）两个字典类型，字典项需自行维护
- 订单表单中的「交易方式」「收款方式」下拉即来自这两个字典

## 操作日志

只读查询页，记录系统中的关键操作。

### 记录范围

- 记录所有新增、修改、删除操作
- 查询类操作不记录；登录相关记录在「登录日志」中查看
- 密码等敏感信息会自动隐藏，不会明文展示

### 搜索与查看

筛选：模块、操作类型、操作人、时间范围。

> 模块和操作类型筛选为精确匹配，如果按条件查不到记录，建议直接按时间范围或操作人查询。

点击「详情」查看：操作内容、操作时间、IP 地址、操作结果（成功/失败）。

## 登录日志

只读查询页，记录每次登录尝试（成功与失败）。

筛选：登录账号（模糊匹配）、登录结果（成功/失败）、时间范围。

列表字段：登录账号、登录 IP、浏览器、登录结果、登录时间。

> 登出和自动续期不记录登录日志。

## 系统参数

集中管理全局配置参数，修改后**立即生效**。

### 预置参数

| 名称 | 默认值 | 作用 |
| --- | --- | --- |
| 默认提成比例（%） | 40 | 销售员未设提成比例时的兜底值 |
| 库存预警阈值 | 0 | 型号未单独设置阈值时的全局预警线 |
| 默认汇率（USD→CNY） | 6.8 | 汇率查询全部未命中时的兜底值 |
| 库存预警邮件通知 | 关闭 | 开启后，库存首次降至预警阈值以下时自动发送邮件通知 |
| 库存预警收件邮箱 | 空 | 接收预警邮件的邮箱地址，多个邮箱用英文逗号分隔 |

> 库存预警邮件需同时满足「开关打开」且「已配置邮箱」才会发送。邮件发送失败不影响正常业务操作。

### 操作

- 搜索：参数名称
- 可新建参数、编辑参数值
- 预置参数不可删除

## 帮助文档管理

管理系统内置的帮助文档内容，支持在线编辑和发布控制。

### 列表与操作

- 搜索：文档标题、分类
- 新建文档、编辑文档、删除文档
- 发布/取消发布：未发布的文档不会在帮助中心展示

### 文档字段

| 字段 | 说明 |
| --- | --- |
| 标题 | 文档标题（必填） |
| 分类 | 文档所属分类，帮助中心按分类分组展示 |
| 内容 | 文档正文，支持图片上传 |
| 关联页面 | 可选，填写对应页面的地址后，用户在该页面点击帮助按钮时会自动打开本文档 |
| 排序号 | 同分类内的显示顺序，越小越靠前 |

> 普通用户通过顶部栏问号按钮或「帮助中心」查看已发布的文档，无需进入此管理页面。

## 个人信息

点击右上角头像 →「个人信息」进入，包含两个 Tab：

### 基本信息

- 查看：用户名（只读）、角色标签
- 可修改：**手机号、邮箱**（姓名和账号不可在此修改）
- 手机号需符合国内 11 位格式，邮箱需符合标准格式

### 修改密码

输入旧密码、新密码（至少 6 位）、确认新密码。

> 修改成功后系统会**强制登出**，需使用新密码重新登录。
