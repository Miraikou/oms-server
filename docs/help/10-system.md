# 系统管理

系统管理模块面向管理员，负责用户、权限、字典与运行配置。

## 用户管理

- 创建/编辑用户，分配角色
- 重置用户密码、启用/停用账号
- 初始账号 admin 由系统自动创建，请及时修改密码

## 角色管理

- 角色是权限的集合，给用户分配角色即授予对应权限
- 内置 SUPER_ADMIN 角色拥有全部权限
- 可为角色勾选菜单权限与按钮权限（如 order:view-all 查看全部订单）

## 菜单管理

- 维护左侧导航的菜单树（目录 / 菜单 / 按钮三种类型）
- 菜单可绑定权限标识与前端路由，控制不同角色可见的功能

## 字典管理

- 维护系统枚举值，如支付方式（PAYMENT_METHOD）、交易方式（TRADE_TYPE）等
- 字典项的增删改即时生效，业务页面下拉选项自动更新

## 系统配置

集中管理全局参数：

- 默认汇率（新建单据自动带入）
- 默认提成参数
- 全局低库存预警阈值（型号未单独设置时生效）

## 日志

- **操作日志**：记录关键业务操作（谁、何时、做了什么）
- **登录日志**：记录登录成功/失败，用于安全审计

## 个人信息

- 右上角头像 →「个人信息」：修改密码、查看自己的角色与权限
