# 快速上手

欢迎使用 OMS 跨境订单管理系统。本文档帮助您在 5 分钟内熟悉系统的基本使用。

> 【图片占位：登录页全貌】

## 登录系统

1. 打开系统地址，进入登录页
2. 输入管理员分配的用户名和密码
3. 点击「登录」进入系统

登录页只有用户名和密码两个输入框，没有验证码。登录成功后系统会自动加载您的菜单权限并进入驾驶舱。

关于登录状态：

- 登录信息会保存在本机浏览器中，关闭浏览器后再打开无需重新登录
- 登录凭证有效期为 7 天，期间系统会自动续期；超过 7 天未使用才需要重新输入密码
- 如果账号被管理员停用，将无法登录，已登录的会话也会在凭证到期后失效

> 首次登录后，建议点击右上角头像 →「个人信息」→「修改密码」修改初始密码。修改密码成功后需要重新登录。

系统预置了超级管理员账号 `admin`（初始密码 `admin123`），请首次部署后立即修改。

## 系统布局

> 【图片占位：主界面布局（侧边栏 + 顶栏 + 标签页 + 内容区）】

- **左侧导航**：按业务模块组织的功能菜单，点击顶栏左侧的折叠按钮可收起/展开。您能看到哪些菜单由管理员分配的角色权限决定
- **顶部栏**：折叠导航按钮、帮助按钮（问号图标）、个人信息与退出登录
- **标签页**：打开过的页面会以标签形式保留在导航下方，支持多页签快速切换
- **内容区**：各功能页面的列表、表单与操作区

### 多标签页与页面缓存

- 系统最多同时缓存 10 个页面，切换标签时页面状态（搜索条件、滚动位置等）自动保留
- 点击标签可回到对应页面；标签上的关闭按钮可关闭页面，**最后一个标签不可关闭**
- 右键点击标签可「刷新当前」（清除缓存重新加载）或「关闭其他标签页」
- 如果某个页面的数据可能过时，右键「刷新当前」即可强制重新加载

## 通用操作约定

系统内所有列表页的操作方式基本一致：

- **查询**：在搜索区填写条件后点击「查询」；「重置」清空条件
- **分页**：表格底部可切换每页条数、跳转页码
- **新增**：列表右上角「新建」按钮，弹窗填写后保存
- **编辑/删除**：表格「操作」列，删除前会二次确认
- **状态开关**：部分列表支持直接切换启用/停用

如果某个按钮您看不到，说明您当前的角色没有该操作权限，请联系管理员在「角色管理」中分配。

## 核心业务流程概览

> 【图片占位：核心业务流程图（订单 → 发货 → 收款 → 完成，采购 → 入库 → 库存支撑）】

OMS 的核心业务围绕「卖得出、发得走、收得回」展开：

1. **销售订单**：与客户达成交易后创建销售订单，订单号以 `SO` 开头
2. **发货**：按订单创建发货单（`FH`），系统按 FIFO（先进先出）自动分配库存批次与成本
3. **收款**：登记客户回款（`SK`，支持美元/人民币多币种）
4. **完成**：全部发货且全部收款后，订单自动流转为「已完成」

采购与库存为销售提供支撑：

- **采购订单**（`CG`）：向供应商采购补货
- **入库**（`RK`）：采购到货后创建入库单，增加库存并生成库存批次（`BT`）
- **库存管理**：查看库存、预警提示、库存调整（`KC`）与流水追溯

售后环节：

- **客户退货**（`TH`）：支持退货退款、退货换货、仅退款、补发不退货四种类型
- **退款**（`TK`）：包括直接退款和退货触发的退款

### 单据编号速查

| 前缀 | 单据 | 前缀 | 单据 |
| --- | --- | --- | --- |
| SO | 销售订单 | FH | 发货单 |
| SK | 收款单 | TK | 退款单 |
| TH | 客户退货单 | CG | 采购订单 |
| RK | 入库单 | PT | 采购退货单 |
| KC | 库存调整单 | BT | 库存批次号 |
| GYS | 供应商编号 | CP | 商品相关编号 |

### 双币种与汇率

系统所有金额均同时记录美元（USD）与人民币（CNY）两个数值：

- 单据上选择币种后，系统按**单据日期**自动查询 USD→CNY 汇率并换算
- 汇率数据来源于「汇率管理」，支持手动维护和外部接口自动同步
- 列表中的金额会同时展示两种币种，主显示币种为单据所选币种

## 权限与角色

- 系统通过「角色」控制权限：管理员给用户分配角色，角色关联可见的菜单和可操作的按钮
- 预置角色包括：超级管理员、老板、销售、采购、仓库、财务
- 超级管理员拥有全部功能；其他角色看到的菜单和按钮各不相同
- 驾驶舱和订单列表支持「我的/全局」视图切换，是否能看到全局数据也由权限控制

## 获取帮助

- 点击顶部栏的 **问号按钮**，会自动打开当前页面对应的帮助章节（例如在订单页点击会直接跳到「销售订单管理」）
- 进入「帮助中心」可浏览全部文档，右侧目录支持快速跳转
- 如遇到权限、账号类问题，请联系系统管理员
