# 快速上手

欢迎使用 OMS 跨境订单管理系统。本文档帮助您在 5 分钟内熟悉系统的基本使用。

## 登录系统

1. 打开系统地址，进入登录页
2. 输入管理员分配的用户名和密码
3. 点击「登录」进入系统

> 首次登录后，建议点击右上角头像 →「个人信息」修改初始密码。

## 系统布局

- **左侧导航**：按业务模块组织的功能菜单，点击展开/收起
- **顶部栏**：折叠导航按钮、帮助按钮（问号图标）、个人信息与退出
- **标签页**：打开过的页面会以标签形式保留，支持多页签快速切换（页面状态自动缓存）
- **内容区**：各功能页面的列表、表单与操作区

## 通用操作约定

系统内所有列表页的操作方式基本一致：

- **查询**：在搜索区填写条件后点击「查询」；「重置」清空条件
- **分页**：表格底部可切换每页条数、跳转页码
- **新增**：列表右上角「新建」按钮，弹窗填写后保存
- **编辑/删除**：表格「操作」列，删除前会二次确认
- **状态开关**：部分列表支持直接切换启用/停用

## 核心业务流程概览

OMS 的核心业务围绕「卖得出、发得走、收得回」展开：

1. **销售订单**：与客户达成交易后创建销售订单
2. **发货**：按订单创建发货单，系统按 FIFO（先进先出）自动分配库存批次与成本
3. **收款**：登记客户回款（支持美元/人民币多币种）
4. **完成**：全部发货且全部收款后，订单自动流转为「已完成」

采购与库存为销售提供支撑：

- **采购订单**：向供应商采购补货
- **入库**：采购到货后创建入库单，增加库存
- **库存管理**：查看库存、预警提示、库存调整与流水追溯

## 获取帮助

- 点击顶部栏的 **问号按钮**，会自动打开当前页面对应的帮助章节
- 进入「帮助中心」可浏览全部文档，右侧目录支持快速跳转
