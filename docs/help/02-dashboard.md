# 驾驶舱

驾驶舱是登录后的默认首页，汇总展示经营全貌。页面自上而下分为五个区块：待处理事项、KPI 指标、进行中订单、趋势分析、排行榜。

> 【图片占位：驾驶舱页面全貌】

## 时间范围与视图切换

页面右上角有两个全局控件：

- **时间筛选**：默认「本月」，可选今天 / 本周 / 本月 / 本季度 / 今年或自定义区间。影响 KPI 与趋势图的统计范围；**待处理事项和进行中订单不受时间筛选影响**（它们统计的是当前实时待办）
- **视图切换**：拥有全局查看权限时，可在「我的业绩」与「全局视图」间切换；无该权限时固定为「我的业绩」。个人视图只统计当前账号关联的销售员数据

> 个人视图依赖「销售员档案」与系统账号的绑定关系（销售员管理 → 关联账号）。如果账号未绑定销售员，个人视图将显示为空。

## 待处理事项

四张待办卡片，点击可直接跳转到对应列表并自动带上筛选条件：

| 卡片 | 统计口径 | 跳转 |
| --- | --- | --- |
| 待发货订单 | 进行中且待发货/部分发货的订单 | 订单列表（自动筛选） |
| 待收款订单 | 进行中且未收款/部分收款的订单 | 订单列表（自动筛选） |
| 待入库采购 | 待入库/部分入库的采购单（仅全局视图显示） | 采购订单列表 |
| 库存预警 | 可用库存 ≤ 预警阈值的商品型号数（仅全局视图显示） | 库存页（自动开启「仅低库存」） |

库存预警卡片右侧会显示「需关注」（红）或「正常」（绿）标签。预警阈值规则详见「库存管理」章节。

## KPI 指标

> 【图片占位：KPI 指标卡片区】

**第一行（销售核心指标，仅统计已完成订单）**：

- **销售额**：已完成订单的总金额合计
- **收款额**：已完成订单的实际收款合计
- **利润**：订单金额 − 博主佣金 − 已退款 − 直接退款 − 产品成本 − 额外成本
- **利润率**：利润 ÷ 销售额 × 100%

标题旁的「订单数」悬停可查看进行中/已完成的单数分布。

**第二行（提成指标，点击跳转提成管理页）**：

- 本月提成（计提合计）、本月冲回、本月净提成、待结算提成

> 提成卡片标题虽写「本月」，实际随页面所选时间范围过滤。

**第三行（仅全局视图）**：采购额（未关闭采购单合计）、库存金额（全部有效批次成本合计，不受时间筛选影响）。

所有金额卡片均同时展示人民币与美元。

## 进行中订单

统计所有进行中（未完成、未取消）订单的实时状况，不受时间筛选影响：

- 订单总额、已收金额、未收金额、预估利润（同利润公式，但针对进行中订单）
- **发货状态分布**：待发货 / 部分发货 / 全部发货各多少单、多少金额
- **收款状态分布**：未收款 / 部分收款 / 已收款各多少单、多少金额

## 趋势分析

> 【图片占位：趋势折线图】

双 Y 轴折线图，横轴按天：

- 销售额（蓝线）、收款额（黄线）、利润（绿线，右轴）
- 全局视图额外显示采购额（紫色柱状，右轴）

悬停可同时查看人民币与美元金额。销售趋势统计进行中+已完成订单；利润与收款趋势仅统计已完成订单。

## 排行榜

**销售员排行**：

- 右上角可切换统计口径：「仅已完成」或「含进行中」（默认含进行中）
- 展示 Top10 销售员的销售额与订单数；个人视图只显示自己一行

**商品销售排行**：

- 可切换「按商品」或「按型号」维度
- 按商品模式下，悬停商品名可查看各型号的件数与金额明细
- 统计口径为进行中+已完成订单，展示 Top10
