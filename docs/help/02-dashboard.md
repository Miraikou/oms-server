# 驾驶舱

驾驶舱是登录后的首页，汇总展示经营关键指标与待处理事项，帮助管理者快速掌握业务全局。

## 待处理事项

页面顶部的四张待办卡片是日常工作的入口，**点击卡片会自动跳转到对应列表页，并预先按相应状态筛选好**：

| 卡片 | 统计口径 | 点击后跳转 |
| --- | --- | --- |
| 待发货订单 | 进行中且未全部发货的订单 | 订单列表（进行中 + 待发货/部分发货） |
| 待收款订单 | 进行中且未全部收款的订单 | 订单列表（进行中 + 未收款/部分收款） |
| 待入库采购 | 待入库/部分入库的采购单 | 采购单列表（待入库/部分入库） |
| 库存预警 | 可用库存低于预警阈值的商品数 | 库存列表（仅低库存） |

> 已取消的订单不计入待发货/待收款；已完成（全部发货且全部收款）的订单自然排除。

## 销售统计

- **销售额/订单量趋势**：按日/周/月/季度维度展示
- **销售员排行**：默认提供「仅已完成订单」的统计口径，避免在途订单造成偏差

## 使用建议

- 每天上班先看待处理事项，按卡片逐项处理
- 库存预警数字是「预警商品数」，点击进入库存页可查看具体商品与型号
