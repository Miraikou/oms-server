# 收款管理

收款管理统一记录客户回款与退款，收款单号以 `SK` 开头，退款单号以 `TK` 开头。

> 【图片占位：收款管理列表页（收款与退款混合展示）】

## 收/退款列表

收款与退款在同一列表中展示，通过「类型」标签区分（收款绿色 / 退款红色）。

### 搜索条件

- 收/退款单号（模糊匹配）、类型（收款/退款）、日期范围

### 列表字段

收/退款单号、类型、订单号、金额（双币种）、币种、汇率、收/付款方式、付款/退款对象、备注、日期。

> 收/退款单提交后**不可修改、不可删除**，请确认金额无误后再提交。如需冲正，请登记一笔反向单据。

## 新建收款

> 【图片占位：新建收款弹窗（含订单金额概览）】

### 入口

- 方式一：在订单列表对目标订单点击「收款」（自动带入订单）
- 方式二：收款管理页点击「新建收款」，手动选择订单（仅列出进行中且未收齐的订单）

### 操作步骤

1. 选择订单后，弹窗展示概览：订单金额、已收金额、**未收金额**
2. 填写收款金额（点击「全部」可一键填入未收金额）
3. 参考汇率自动按收款日期查询（只读，仅供核对）
4. 填写收款日期（默认当天）、收款方式（可选，在字典管理维护）、付款方、备注
5. 提交

### 规则说明

- **收款币种固定为订单币种**，不可选择其他币种
- 累计收款不能超过订单总金额，超出时系统拒绝提交
- 汇率按收款日期的 USD→CNY 汇率自动换算双币种入账
- 收款成功后订单已收金额累加，收款状态随之更新

### 收款冲抵直接退款

如果该订单此前发生过直接退款，本次收款会**优先冲抵退款金额**：冲抵后订单利润相应恢复，提成也会差额重算。

## 新建退款

> 【图片占位：新建退款弹窗】

### 入口

- 方式一：在订单列表对目标订单点击「退款」
- 方式二：收款管理页点击「新建退款」，手动选择订单（列出所有有已收款的订单）

### 操作步骤

1. 选择订单后，弹窗展示概览：订单金额、已收金额、**可退金额**（= 已收金额）
2. 填写退款金额（不能超过可退金额）
3. 参考汇率自动查询（只读）
4. 填写退款日期、退款方式、退款对象、备注
5. 提交

### 规则说明

- 已取消的订单无法退款
- 累计退款（含客户退货退款）不能超过已收金额
- **退款使用订单原始汇率换算**（而非退款日汇率），保证「全额退款 → 利润归零」不产生虚增汇兑损益
- 退款会扣减订单已收金额，收款状态可能回退（如已收款 → 部分收款）

## 对订单状态的影响

- 收款累加已收金额：未收款 → 部分收款 → 已收款
- 直接退款扣减已收金额：状态可能反向回退
- 全部发货 + 已收款 → 订单自动「已完成」
- 已完成订单因退款导致收款状态下降 → 订单重新打开为「进行中」

> 注意区分：客户退货触发的退款（退货退款/仅退款）**不扣减已收金额**，而是记入独立的「退货退款」字段，不影响收款状态。

## 对提成的影响

- 收款使订单变为「已完成」→ 自动计提销售提成
- 退款使订单从已完成退回进行中 → 已计提提成**全额撤销**
- 退款后订单仍为已完成 → 提成按新利润**差额重算**（可能冲回，也可能因冲抵后利润上升而补提）

## 查看收/退款记录

收款管理列表本身没有详情页。要查看某订单的完整收退款历史，请打开订单详情抽屉，其中包含「收款记录」和「退款记录」两个表格。
