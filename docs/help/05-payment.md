# 收款管理

收款单登记客户的每一笔回款，支持美元/人民币多币种，并自动冲抵订单应收。

## 登记收款

收款基于销售订单创建：

1. 在订单列表对目标订单点击「收款」
2. 选择收款币种（USD / CNY）
3. 填写本次收款金额、收款方式（银行转账、PayPal 等，见字典配置）
4. 可填写备注（如流水号），提交

## 收款状态联动

- 订单累计已收金额达到应收金额后，收款状态变为「已收款」
- 部分回款时状态为「部分收款」，可多次登记
- 全部发货 + 全部收款 → 订单自动「已完成」

## 收款单列表

- 支持按订单号、收款币种、日期范围筛选
- 列表展示每笔收款的币种、金额、折算人民币金额与备注

## 退款

- 发生退款时，在订单操作中使用「退款」功能，填写退款金额与币种
- 退款会冲减已收金额，收款状态随之回退

> 注意：已有收款的订单不能直接取消，需先完成退款。
